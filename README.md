# Funnel Recorder

A tool to open any URL and record user interactions (clicks, inputs) using Puppeteer.

## Usage

1. Install dependencies:
   ```
   npm install
   ```

2. Run the server:
   ```
   npm start
   ```

3. Visit:
   ```
   http://localhost:3000
   ```

4. Enter the URL you want to record.

5. A browser will open. Interact with the page.
